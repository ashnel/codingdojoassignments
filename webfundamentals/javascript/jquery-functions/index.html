.hide/.show

Click the ".hide" button to hide the paragraph. Click the ".show" button to make it reappear!

.toggle

Click the ".toggle" button to hide the paragraph. Click the ".toggle" button again to make it reappear!

.slideUp/.slideDown

Click the ".slideUp" button to hide the paragraph by sliding it up. Click the ".slideDown" button to make it reappear by sliding it back down!

.slideToggle

Click the ".slideToggle" button to slide the text up. Click the ".slideToggle" button again to make it reappear by sliding down!

.fadeIn/.fadeOut

Click the ".fadeIn" button to make the text slowly appear. Click the ".fadeOut" button to make the text slowly disappear!

.addClass

Click the ".addClass" button to add a CSS class that will change the color of the text to red!

.before/.after

Click the ".before" button to make a new H2 appear before the paragraph. Click the ".after" button to make a new H2 appear after the paragraph!

Click the ".append" button to add something to the end of this paragraph.
